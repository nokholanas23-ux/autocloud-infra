# 🚀 AutoCloud Infra

> Automated cloud infrastructure for small businesses — deploy a complete, secure cloud environment in minutes.

## 📋 What This Does

This project automatically deploys a production-ready cloud infrastructure on Azure using Terraform and GitHub Actions CI/CD pipeline.

## 🛠️ Technologies Used

- **Terraform** — Infrastructure as Code
- **Microsoft Azure** — Cloud Provider
- **GitHub Actions** — CI/CD Pipeline
- **Docker** — Containerization
- **Prometheus** — Monitoring
- **Grafana** — Dashboard & Alerts

## 🏗️ Infrastructure Components

- ✅ Resource Group
- ✅ Virtual Network & Subnet
- ✅ Linux Virtual Machine (Ubuntu)
- ✅ Network Security Group
- ✅ Public IP Address
- ✅ Monitoring Dashboard (Grafana)

## 🚀 How to Deploy

1. Clone this repository
2. Configure Azure credentials
3. Run Terraform:
```
terraform init
terraform plan
terraform apply
```

## 📊 Monitoring

Live Grafana dashboard showing:
- Server uptime
- CPU & memory usage
- Network traffic
- Custom alerts
